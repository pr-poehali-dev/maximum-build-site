┌───────────────────────────────────────────────────────────────────────┐
│                    ФАЙЛЫ И ИХ РОЛИ                                    │
└───────────────────────────────────────────────────────────────────────┘

    index.html
    ────────────────────────────────────────
    • Главный HTML файл
    • Загружает JavaScript и CSS
    • Содержит <div id="root">
    • Все маршруты возвращают этот файл

    .htaccess
    ────────────────────────────────────────
    • Конфигурация Apache
    • Перенаправляет все запросы на index.html
    • Исключение: существующие файлы
    • Включает GZIP и кэширование

    assets/index-[hash].js
    ────────────────────────────────────────
    • Весь JavaScript код приложения
    • Минифицированный и оптимизированный
    • React, React Router, компоненты
    • Хеш в имени для кэш-busting

    assets/index-[hash].css
    ────────────────────────────────────────
    • Все стили приложения
    • Tailwind CSS классы
    • Минифицированный
    • Хеш в имени для кэш-busting

Процесс от начала до конца:

0 мин   │ Скачивание проекта
        ▼
2 мин   │ npm install (установка зависимостей)
        ▼
5 мин   │ npm run build (сборка)
        ▼
6 мин   │ Копирование файлов на сервер
        ▼
8 мин   │ Настройка Apache (mod_rewrite)
        ▼
10 мин  │ Настройка домена (опционально)
        ▼
12 мин  │ Проверка в браузере
        ▼
15 мин  │ ✅ САЙТ РАБОТАЕТ!

Итого: 15 минут (при наличии опыта может быть 5-10 минут)
